Git Cheat Sheet

Common Git Commands

Command Description Documentation
git init Initialize a new Git repository Git Init
git status Check the status of the working directory Git Status
git add Stage changes for the next commit Git Add
git commit Commit staged changes to the repository Git Commit
git push Push committed changes to a remote repository Git Push
git pull Fetch and merge changes from a remote repository Git Pull
git clone Clone an existing repository Git Clone
git log View the commit history Git Log
